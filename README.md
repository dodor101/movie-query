# Description

### Movie Query is a simple application that allows you to query movies by their titles. It uses that OMDB API. By searching for a movie you will be presented with with specific info about that movie.

## Technologies Use

- jQuery
- JavaScript
- TailwindCSS
- OMDB (API)
